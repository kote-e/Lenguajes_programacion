# Carrera de Tortugas 🐢
Bienvenido/a a la experiencia más intensa, caótica y completamente injusta del mundo de las carreras…
una carrera de tortugas con apuestas

## Explicacion
Este proyecto simula una carrera de tortugas donde:

- Cada tortuga avanza de forma aleatoria 🐢
- Pueden ocurrir eventos caóticos (ají, hongos, catapultas, etc) 🌶️🍄
- Puedes apostar dinero 💸


## Instalación
pip install -r requirements.txt

## Ejecución
python carrera.py

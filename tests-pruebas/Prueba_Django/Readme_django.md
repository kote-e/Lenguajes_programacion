# Proyecto Django - Guía de Inicio
## 1. Crear un entorno virtual

### Crear entorno virtual:

``` python -m venv venv ```

### Activar entorno virtual
Windows (CMD)
``` venv\Scripts\activate ```
Windows (PowerShell)
``` .\venv\Scripts\Activate.ps1```

### Desactivar entorno virtual
``` deactivate ```
